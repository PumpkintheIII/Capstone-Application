This is a demonstration of what this could look like. While this is not close to what I think a final product would look like, it is hopefully helpful to see what it could look like.

This Home page would contain basic information such as:

  • What the project is.
  • Why it is important.
  • How to use it.

How to use this

Here is a brief explanation of this application:

  • On your left you have a list of pages of information. By pressing each of them, you can navigate around the application to learn different aspects of fire building. While I have currently only included a few basic topics, there is much more to fire building that could be added in the future.
  • On the top of the page, you have a header that tells the user what page they are on. This gives the user a general idea of what they will learn in this part of the application.
  • Below the header, there is a scrollable list of cards that contain the information the application intends to teach.
  • At the bottom of the page is a button to move onto the next page. This allows a user to move onto the next part of the application without using the navigation bar.
